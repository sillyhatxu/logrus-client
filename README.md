# logrus-client


## Initialize your project

```
go mod init logrus-client
```

## Updating dependencies

```
go mod vendor
```

## verify dependencies

```
go mod verify
```

## remove dependencies that is not used

```
go mod tidy
```

## print dependence diagram

```
go mod graph
```

## download dependencies

```
go mod download
```
